How to edit logo & favicon?

Warning! Your logo should be less than 500px. My recommendation is that the height should be 50px and the width any. Delete the empty space between the logo type, otherwise your logo will be very small!

Logo you can add direct via WHMCS Admin.So go to yoursite.com/admin and login.After Login find in right top corner this icon and next step find Setup Wizard.Open this press Next and find Logo press Browse and upload your logo!

How to edit Headbar and Navbar?

Headbar you can find in WHMCS_ROOT/templates/ruzenko/header.tpl and you can edit e-mail adress or adding some buttons or etc.

Navbar or Navigation is not possible to edit easy and there are 2 ways:

  • Manual with hooks WHMCS_ROOT/includes/hooks/ check this link for documentation.
  • Automatic with Advanced Navbar Manager.

After you save this file go to WHMCS_ROOT/templates/ruzenko and create create example minecraft.tpl.

After this part go to WHMCS_ROOT/templates/ruzenko/includes/pageheader.tpl preview and there add under all this {else if $templatefile == 'minecraft'} save and close.Go to WHMCS_ROOT/templates/ruzenko/header.tpl preview same like in pageheader.tpl paste {else if $templatefile == 'minecraft'} and that it go to yoursite.com/minecraft.php THIS IS ONLY EXAMPLE minecraft page already exists.
